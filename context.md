# Auth service

## Tech usage:
- BE: Golang
- Cache: Redis
- DB: Postgres
- FE: Vue 3
- Cloud and ci/cd: GCP (secret manager, cloud run, ...), github action.
## Features:

- Login with gg
- Login with email and password
- Manage roles and permissions
- Enable 2FA (can use Password (ios) or Authentication(google))
